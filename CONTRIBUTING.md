# Contributing

Thank you for contributing to the FalkorDB Claude Code skills file.

## Guidelines

- Keep changes practical and specific to FalkorDB.
- Provide explicit, runnable examples for each skill.
- Avoid marketing language; use concise, authoritative tone.
- Prefer official documentation sources and cite exact commands.
- Validate that examples use correct FalkorDB commands and syntax.

## Submitting changes

1. Make edits in [skills.md](skills.md).
2. Ensure examples are minimal and accurate.
3. Open a pull request with a clear summary of changes.
