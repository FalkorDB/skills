# Operations Skills (Docker)

This directory contains 6 individual skills related to FalkorDB Docker operations and deployment.

## Skills

1. **run-with-browser-for-dev** - Run FalkorDB with browser UI for local development
2. **run-server-only** - Run server-only container for production
3. **set-authentication** - Configure authentication using REDIS_ARGS
4. **set-module-config** - Tune FalkorDB settings with FALKORDB_ARGS
5. **use-docker-compose** - Create repeatable stacks with docker-compose.yml
6. **run-browser-separately** - Run browser and server as separate containers

## Usage

Each skill is in its own directory with a `skill.md` file that includes:
- YAML frontmatter with name and description
- Usage instructions
- Example code
- Important notes

## About Operations

These skills focus on deploying and configuring FalkorDB using Docker, covering everything from local development setups to production-style deployments with proper authentication and configuration.
